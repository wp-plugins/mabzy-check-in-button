== Description ==


The Mabzy Check-in button allows you to check-in to the current post for Mabzy. Mabzy allows you to share websites with your friends as well as spread the word about cool websites to help website owners.

= New =

* Nothing

= Features =

* Compact or Large Button View
* Apply CSS to the container.
* Integrates with Wordpress MU

== Installation ==

Follow the steps below to install the plugin.

1. Upload the Mabzy directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Mabzy in the navigation to configure the button
